# Chapter 3 – Review (Pros, Cons, Improvements)

**Topic:** Electricity – uses, sources, circuits, torch, cell, battery, lamps, switch, symbols, conductors/insulators  
**Files:** 10 simulations (sim1 electricity uses through sim10 conductors insulators).

## Pros

- Progressive build from uses to sources to components to cell, battery, lamps, circuit, switch, symbols, conductors.
- Consistent controls; English identifiers (toggleSwitch, setSwitchType, testMaterial); Kannada labels for students.
- Visual feedback: bulb glow, wire animation, switch states, conductor/insulator counts.
- Safety messaging in sim10 (e.g. body as conductor, safety rules).

## Cons

- Some sims dense; optional step mode could help.
- Buttons and interactive areas lack aria-labels.

## Improvements

1. Add aria-label to key controls (e.g. switch on, select material).
2. Optional: One-sentence "what you learn in this sim" at top.
3. Optional: Previous/next link strip for chapter navigation.
